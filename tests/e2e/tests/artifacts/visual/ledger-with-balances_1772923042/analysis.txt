# Detailed Analysis of APEX MCD Ledger Interface

## Overall Layout & Structure
- **Header Navigation**: Dark banner with tabs (APEX MCD, SIMULATE, TRANSFERS, REVIEW QUEUE, LEDGER, SETTLEMENT, RETURNS) and "SYSTEM NOMINAL" status indicator on the right
- **Main Content Area**: Single table-based view displaying account balances
- **Color Scheme**: Dark background (near black #1a1a1a) with green text highlights and red accent colors

## Typography & Text Hierarchy
- **Font**: Monospace system font (appears to be a terminal/code style font like Courier or similar)
- **Sizes**: Consistent small-to-medium sizing throughout
- **Colors**: 
  - White text for primary content
  - Bright green (#00FF00 or similar) for positive balances
  - Red (#FF0000 or similar) for negative balances
  - Muted gray for secondary labels

## Header Section
- "LEDGER" title in white, bold serif font
- "ACCOUNT BALANCES" subtitle in smaller gray text

## Visual Styling Details
- **Borders**: Green left border on table (indicating active/valid state)
- **Spacing**: Compact row spacing with consistent padding
- **Balance Alignment**: Right-aligned currency values
- **Negative Value Indicator**: Red text color with minus sign prefix
- **Row Alternation**: Subtle or no visible alternating row colors
- **Hover States**: Not visible in static screenshot

## Financial Logic
- Balances sum to zero: $250.00 (INV-1801) - $250.00 (OMNI-ACME) = $0.00
- Indicates a balanced ledger system with omnibus accounts offsetting investor positions
